Below is the list of all commands. /about - view version and developer information /add - add a new validator to the chat, optionally followed by the address /contact - send a bug report or feature request /help - view the list of all commands /networkstatus - view the current network status information, alias /network /nominations - view a summary of nominations, alias /n /nominationdetails - view nomination details, alias /nd /payouts - view monthly nominator payouts report /referenda - view referenda being voted and their details /remove - remove a validator from the chat /rewards - view monthly validator rewards (i.e. income) report /settings - configure notifications /validatorinfo - view detailed validator information, alias /vi
